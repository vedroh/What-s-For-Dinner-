# What-s-For-Dinner-
Проект по ML, в рамках которого мы создаем web-приложение, которое: 

1) Оценивает магазинный чек пользователя, считывает купленные продукты.
2) Предлагает рецепты, которые можно реализовать с купленными продуктами и присваивает каждому приоритет, в зависимости от того, сколько есть продуктов из нужных для выбранного блюда.
